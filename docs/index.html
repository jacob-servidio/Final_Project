Story

The purpose of this project is to create a series of interactive visuals that compare total Olympic medal counts with global geography and Average temperature. This project utilizes (2) databases, 1) 120-year complete history of Olympic participation (both Summer and Winter), 2) Long-Run Global Temperatures for all countries. The (3) visuals created were a bubble chart, a Global Map, and a stacked-bar chart.
